# Solana Takoyaki

## Overview

This is a proxy service that exposes a SubQuery Data Node RPC spec to query filtered blocks from Solana via the SQD Soldexer.

## Building

To generate a binary:
```bash
make build
```

To run:
```bash
make run
```

## Options

```
  -port uint
    Port to listen on (default 8080)
```
